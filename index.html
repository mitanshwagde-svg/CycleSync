System Architecture Overview.

Inputs

Dirt Level and Load Size are modeled on a 0-10 universe with overlapping fuzzy sets.

02

Inference

A Mamdani rule base blends linguistic states instead of relying on brittle threshold logic.

03

Output

Wash Time is defuzzified over a 0-60 minute range to produce smooth machine recommendations.

Membership Functions

Triangular fuzzy sets for dirt level, load size, and wash time.

Response Surface

The output rises as dirt and load increase, which matches real washing behavior.
